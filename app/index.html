Filters even/odd problem

Simple list:

  1. Odd element
  2. Even element

Mistake

jQuery starts counting from 0, when catch even or odd numbers.
So first item (index = 0) is divided by 0, so is even. Second item (index = 1) is not divide by 2, so is odd.

CSS selectors starts counting from 1.
So first item (index = 1) is not divided by 2, so is odd. Second item (index = 2) is dividev by 0, so is even.
